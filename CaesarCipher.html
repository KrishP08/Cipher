Welcome to the Caesar Cipher Tool!

The Caesar cipher is a simple substitution cipher where each letter in the plaintext is shifted a certain number of places down or up the alphabet.

To Encrypt a Message:

  1. Enter a Message: Type the text you want to encrypt into the "Message" field.
  2. Set a Shift Value: Choose an integer for the "Shift Value." This is your key.
  3. Click Encrypt: The encrypted message will appear in the "Result" box.

To Decrypt a Message:

  1. Enter the Ciphertext: Paste the encrypted text into the "Message" field.
  2. Use the Same Shift Value: Enter the original key used for encryption.
  3. Click Decrypt: The original message will be revealed.

Tip: You can use the "Show/Hide Alphabet Tables" button to see how the alphabet is shifted based on your key.
